# Stripe Integration

:::warning Development Status
Stripe integration is currently in development and will be available in a future release.
:::

## Planned Features

The upcoming Stripe implementation will include:
- One-time payments
- Subscription management
- Usage-based billing
- Webhook handling
- Customer portal integration
- Invoice management

## Current Status

While Stripe integration is being developed, you can:
1. Set up your Stripe account
2. Gather your API keys
3. Plan your pricing structure
4. Design your subscription tiers

:::info Stay Updated
Follow our [GitHub repository](https://github.com/eliasnau/starter-kit) for updates on Stripe integration and other payment features.
:::
