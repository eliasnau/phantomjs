# Final Steps

## Database Setup (Optional)

<details>
<summary>If you need to set up a local PostgreSQL database</summary>

You can start a PostgreSQL database using the provided script:

```bash
./start-database.sh
```

This will create a Docker container with PostgreSQL installed and configured according to your environment variables.
</details>

## Populate your database

Push the database schema to your PostgreSQL instance:

<Tabs>
<TabItem value="pnpm">

```bash
pnpm db:push
```

</TabItem>
<TabItem value="npm">

```bash
npm run db:push
```

</TabItem>
<TabItem value="yarn">

```bash
yarn db:push
```

</TabItem>
</Tabs>

## Start the local server

Start your development server:

<Tabs>
<TabItem value="pnpm">

```bash
pnpm dev
```

</TabItem>
<TabItem value="npm">

```bash
npm run dev
```

</TabItem>
<TabItem value="yarn">

```bash
yarn dev
```

</TabItem>
</Tabs>
